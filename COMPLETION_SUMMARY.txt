## 🎨 Customization Examples

### Change Accent Color
Edit `tailwind.config.js`:
```javascript
accent: '#00ff00' // Change to green
```

### Add New Language
Edit `src/i18n.js`:
```javascript
fr: {
  nav: { services: 'Services' },
  // ... add all translations
}
```

### Update Company Info
Edit `src/i18n.js` contact section

### Modify Services
Edit `src/components/Services.jsx` or `src/i18n.js`
